Política de Privacidad

Esta política describe cómo “Tu Empresa” (la “Aplicación”) recopila, usa y protege tu información personal cuando utilizas el inicio de sesión y las funcionalidades del sistema.

Datos que recopilamos

  • Correo electrónico
  • Nombre para mostrar
  • Foto de perfil (si el proveedor la suministra)
  • Identificador único de usuario (UID)

Cómo usamos los datos

Usamos esta información exclusivamente para:

  • Autenticar tu acceso a la Aplicación.
  • Mantener tu sesión y perfil básico de usuario.
  • Mejorar la seguridad y prevenir accesos no autorizados.

Almacenamiento y seguridad

Los datos se almacenan de forma segura en Firebase Authentication y Firestore. Implementamos medidas técnicas y organizativas adecuadas para proteger tu información.

Compartición de datos

No vendemos ni compartimos tu información personal con terceros, salvo los proveedores de autenticación (por ejemplo, Google, GitHub, Facebook) que utilizas para iniciar sesión.

Tus derechos

Puedes solicitar el acceso, actualización o eliminación de tus datos. Consulta la sección Eliminación de datos de usuario para más información.

Contacto

Si tienes preguntas sobre esta política, escríbenos a: soporte@tuempresa.com.
